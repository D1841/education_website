# STUDENT WEBS - Educational Website

## Project Overview

**STUDENT WEBS** is a comprehensive educational website designed to support students in Ghana with academic resources, learning materials, and educational guidance. The platform provides access to various courses, learning resources, and links to reputable educational websites.

---

## Website Information

- **Website Name**: STUDENT WEBS
- **Purpose**: Educational support platform for students
- **Target Audience**: Students in Ghana (all educational levels)
- **Launch Year**: 2026
- **Status**: Active and fully functional

---

## How to Use

### For Students
1. Visit the **Home** page to get an overview of STUDENT WEBS
2. Check the **About** page to learn more about the platform
3. Browse **Courses** to find your subject of interest
4. Click on course links to access external learning platforms
5. Use the **Resources** page to find comprehensive study materials
6. Fill out the **Contact** form for any inquiries or feedback

### For Deployment
1. Ensure all files are in the same directory
2. All image files must be in the same folder as HTML files
3. Verify CSS file is properly linked in each HTML page
4. Test all external links for accessibility
5. Test responsive design on various devices
